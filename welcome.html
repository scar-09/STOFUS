STOFUS

What Stofus Does:

Blocks websites and keywords. You choose what.

Block Types

  • Schedule Block: blocks a site during specific hours every day.
  • Timer Block: blocks a site for a fixed duration with no bypass.

Removing Blocks

  • Schedule blocks can be removed through the friction page.
  • Timer blocks expire automatically.
  • Keywords can be removed instantly from the popup.

Limitations

  • Stofus only works in this browser. It does not block other browsers or devices.

Incognito Protection:

Stofus cannot block incognito tabs unless permission is enabled.
